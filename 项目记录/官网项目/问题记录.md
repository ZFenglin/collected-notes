## vite别名设置

vite.config.ts

```JS
export default defineConfig({
    resolve: {
        alias: {
            // '@': '绝对路径'
            '@': path.join(__dirname, './src')
        }
    }
})
```

tsconfig.json

```JSON
  "compilerOptions": {
    // ...
    "baseUrl": "./",
    "paths": {
      "@": [
        "src"
      ],
      "@/*": [
        "src/*"
      ]
    }
  },
```
